    Flight Plan / FMC Setup
    Introduction
  • The Starship comes equipped with two means of navigation:
  • The FMC does not support GPS or SID/STAR's. Use SimBrief's Legacy Algorithm 👁 to find a compatible route.
  • The GNS 430 👁 supports GPS and SID/STAR's.
  • The GNS can be controlled by PMS5's free GTN750 widget 👁. You can download it by visiting pms50.com/msfs
    • Init
    • Pilot and Copilot AvionicsON
    • CDU → INITIALIZE SYSTEM → ALL DATA OK
    • MSG (btn) → UPDATE VLF → UPDATE VLF FROM FMS
    • SYS CTRL (btn) → Verify AUTO LEG is active → Verify AP NAV SRC is FMS or GNS
    • NAV SOURCE (btn) → Verify FMS 1 or FMS 2
    • Request clearance (IFR)
      • NAV DATA Update
    • Insert floppy disk 👁
    • UPDATE DATA BASE → Select FLIGHT DATA → CONTINUE
    FMS-850
    • When using SimBrief VOR route
    • MSFS EFB → SimBrief Dispatch → VIEW → IMPORT ROUTE → FLT PLAN (btn)
    • Click Departure airport → REPLACE WPT → Enter departure airport → DISPLAY RUNWAYS
    • MFD Display → PLAN MAP → DISPLAY → Verify Route
    • When not using SimBrief
    • FLT PLAN → Click Screen to activate/deactivate hardware keyboard → Enter departure airport, waypoints, arrival airport → END FPL
    • (optional) FPL → APPR → Set Approach
    • MFD Display → PLAN MAP → DISPLAY → Verify Route
    • To delete a flight plan, click on any WPT and select ERASE FLPT
    • (optional) VNAV (btn) → Set altitudes (advisory only)
    GNS
    • It is highly recommended to use the free GNT750 widget to set up the GNS
    • Open GNT 750 Widget
    • Flight Plan → MENU → Import → Import from SimBrief → Back
    • PROC → Departure → Select SID/TRANS → Load Departure → Back
    • (optional) PROC → Arrival → Select STAR/TRANS → Load Arrival → Back
    • (optional) PROC → Approach → Select runway → Load Approach → Back
    • Flight Plan → Verify RTE
