Cessna 172S Skyhawk Procedures

Cessna 172S Skyhawk
Pre-Flight
Exterior Inspection
  • Remove control wheel lock CHECK
  • Check fuel quantity and quality VERIFY
  • Inspect wings and surfaces CHECK
  • Check tire condition and inflation VERIFY
  • Remove pitot tube cover REMOVE
Interior Inspection
  • Check parking brake set SET
  • Ensure master switch OFF VERIFY
  • Check ignition switch OFF CHECK
  • Verify fuel selector on BOTH CONFIRM
  • Inspect controls for freedom of movement CHECK
Before Start
Cockpit Preparation
  • Seat and harness adjusted SET
  • Radios and avionics OFF VERIFY
  • Circuit breakers IN CHECK
  • Doors and windows CLOSED SECURE
  • Mixture RICH SET
  • Throttle set (1/4 inch open) ADJUST
  • Primer locked CHECK
Before Taxi
Engine Start
  • Master switch ON ON
  • Beacon ON ON
  • Fuel pump ON ON
  • Throttle cracked (1/4 inch) SET
  • Mixture rich SET
  • Clear area and call "CLEAR" CLEAR
  • Ignition START ENGAGE
  • Oil pressure checked CHECK
  • Fuel pump OFF OFF
  • TABLET: Set engine parameters
  • ATC: Request engine start
Before Takeoff
Run-up Checks
  • Brakes checked TEST
  • Flight instruments set SET
  • Engine run-up (1700 RPM) PERFORM
  • Magnetos checked (max drop 150 RPM) CHECK
  • Carb heat checked (RPM drop) TEST
  • Vacuum pressure checked VERIFY
  • Engine idle checked CHECK
  • Flight controls FREE and CORRECT VERIFY
  • ALT: Set appropriate altitude for departure
  • TABLET: Verify flight plan loaded
  • ATC: Request takeoff clearance
Cruising
Climb Checks
  • Flaps UP RETRACT
  • Landing light OFF (above 10,000 ft) OFF
  • Mixture leaned ADJUST
  • Fuel selector on BOTH SET
  • Engine gauges in green MONITOR
  • ALT: 6500 feet
  • TABLET: Monitor engine performance
Cruise Checks
  • Throttle set for cruise power SET
  • Mixture adjusted for altitude ADJUST
  • Fuel pump OFF (if not required) OFF
  • Avionics and systems check VERIFY
  • Passenger comfort check CHECK
  • ALT: 7500 feet
  • TABLET: Check weather and navigation
  • ATC: Request higher altitude if needed
